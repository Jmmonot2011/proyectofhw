En esta sección se mostrarán las aplicaciones de mantenimineto y recuperación de datos y como descargarlas e instalarlas siguiendo una serie de tutoriales
MAC OS-Windows

DiskDrill

Disk Drill es un programa para la recuperación de datos eliminados, compatible tanto con ordenadores con Windows como con Mac OS
Ubuntu-Windows

TestDisk

TestDisk es un poderoso software gratuito de recuperación de datos! Fue principalmente diseñado para ayudar a recuperar particiones perdidas y/o volver discos no booteables a booteables nuevamente cuando estos síntomas son causados por software con fallas, ciertos tipos de virus o error humano
