Microsoft Word

•Microsoft Windows 7 SP1, Windows 8, Windows Server 2012, Windows 10, Ubuntu 20.04(32/64)
•Windows Server 2008 32/64 bits, Windows 7 32/64 bits, Windows 8 32/64 bits o Windows 10
•PC Pentium compatible (Pentium III, Athlon, aunque se recomienda un sistema más reciente)
•Procesador de 1 GHz o más rápido con el conjunto de instrucciones de SSE2.
•256 MB de RAM (se recomiendan 512 MB de RAM)
•1 GB o más de memoria RAM para 32 bits, 2 GB o más de memoria RAM para 64 bits.
•Al menos 1,5 GB disponibles en el disco duro
•Espacio disponible en disco duro de 3 GB o más
•Resolución de por lo menos 1024*768 (se recomienda más alta), con al menos 256 colores.
•Resolución de pantalla de 1024 x 576 con Directx10.
